# Enterprise Payment Dashboard

A modern React-based payment management dashboard built using **React**, **TypeScript**, **Material UI**, and **Axios**. The application integrates with the Enterprise Payment API to provide a clean, responsive interface for managing payment transactions.

> **Backend Repository:** https://github.com/vinit-joshi1989/enterprise-payment-api

---

## Features

- Dashboard with payment summary cards
- Search by Transaction Reference, Customer ID, or Description
- Filter payments by Status
- Sort payments by:
  - Created Date
  - Amount
  - Transaction Reference
- Client-side Pagination
- Create New Payment
- Update Payment Status
- Delete Payment with Confirmation
- Success and Error Snackbar Notifications
- Responsive Material UI Interface

## Getting Started

### Prerequisites

- Node.js 20+
- npm
- Enterprise Payment API running locally

### Clone Repository

```bash
git clone https://github.com/vinit-joshi1989/enterprise-payment-dashboard.git
cd enterprise-payment-dashboard
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```
